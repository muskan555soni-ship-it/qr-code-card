# QR Code Component

This is my solution to the QR Code Component challenge from Frontend Mentor.  
The goal was to build a simple and responsive card layout using HTML and CSS.

---

## Overview

This project displays a QR code card with a heading and description.  
It helped me practice layout design, alignment, and basic styling.
